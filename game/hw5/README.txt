Four libraries were apt installed for this project:
libzmq3-dev, libzmqpp-dev, nlohmann-json3-dev, and libnode-dev

Parts 1 and 2 of the assignment are in separate directories.
Part 1 is in hw5/platformer, part 2 is in hw5/flappy
To compile and run the project for each part,
navigate to the respective directory and run make && ./main

platformer controls:
w to jump
a d to move left and right
w and shift at the same time to die

Flappy Bird controls:
w to flap
